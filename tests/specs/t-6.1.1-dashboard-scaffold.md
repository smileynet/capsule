## Test Cases

| Input | Expected Output | Notes |
|-------|-----------------|-------|
| Render priority badge P0 | Red colored "P0" text | Critical priority |
| Render priority badge P3 | Blue colored "P3" text | Normal priority |
| Focused border style | Accent-colored rounded border | Visual focus indicator |
| Unfocused border style | Dim rounded border | Non-focused pane |
| BrowseKeyBindings() | Contains up, down, enter, tab, r, q | Browse mode keys |
| PipelineKeyBindings() | Contains up, down, tab, q | Pipeline mode keys |
| SummaryKeyBindings() | Contains "any key" | Summary mode keys |
| cache.Get("id") on empty | nil, false | Cache miss |
| cache.Set("id", detail) then Get | detail, true | Cache hit |
| cache.Invalidate() then Get | nil, false | Cache cleared |
